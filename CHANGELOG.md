# Changelog

## v0.2.1 - 2026-03-24

### 新增
- 新增统一应用内弹窗与轻提示，替换原生 `alert / confirm / prompt` 体验。
- 新增 AI 自定义试卷、AI 自定义练习的本地持久化保存能力。
- 新增用户数据导入能力，可恢复历史记录、练习进度与个人配置。
- 新增成就广场扩展展示与更多成长激励内容。
- 新增桌面端自动更新检测与 GitHub Release 安装包分发链路。

### 优化
- 重构多处核心页面布局，减少空白与多余卡片，统一视觉密度与交互节奏。
- 优化暗色 / 亮色模式配色，降低刺眼感，统一护眼色调。
- 优化引导页、头像区、成就区、历史记录区与试卷操作区的排版细节。
- 优化应用 Logo、安装包图标与桌面端整体品牌呈现。
- 优化分享、导出、保存、删除、更新检查等反馈方式，统一为应用内风格。

### 修复
- 修复多个页面深色模式适配不完整的问题。
- 修复试卷页 PDF / 开始答题 / 分享按钮尺寸与对齐不一致的问题。
- 修复成长日历颜色反馈不明显、悬浮信息缺失等问题。
- 修复筛选、历史对比、成就展示、头像区域等若干布局与交互异常。
- 修复 AI 生成页与答题页残留原生确认框、状态提示不统一的问题。

### 数据
- 清理与重构题库资源组织方式，支持题目图片资源本地化引用。
- 保留题库内容的同时支持重置用户初始状态，便于新用户开箱体验。

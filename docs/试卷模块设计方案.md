# OpenExam 试卷模块设计方案

## 一、模块概述

对标 SaDuck 公考知识库，设计完整的试卷练习系统。

## 二、核心功能模块

### 1. 历年真题
| 功能 | 说明 |
|------|------|
| 试卷列表 | 按年份、考试类型（国考/省考）分类展示 |
| 试卷筛选 | 支持年份、地区、科目筛选 |
| 试卷详情 | 显示题目数量、预计时长、难度等级 |

### 2. 专项练习
| 功能 | 说明 |
|------|------|
| 分类练习 | 言语理解、数量关系、判断推理、资料分析、常识判断 |
| 知识点标签 | 每道题关联知识点，便于针对性练习 |
| 智能推荐 | 根据错题自动推荐薄弱知识点 |

### 3. 练习记录
| 功能 | 说明 |
|------|------|
| 做题历史 | 记录每次练习的时间、得分、用时 |
| 错题本 | 自动收集错题，支持重做 |
| 统计分析 | 正确率趋势、知识点掌握度 |

## 五、技术实现要点

### 5.1 前端组件
- `PaperList` - 试卷列表组件
- `PaperFilter` - 筛选器组件
- `ExamRoom` - 答题界面组件
- `QuestionCard` - 题目卡片组件
- `AnswerSheet` - 答题卡组件
- `ResultPage` - 成绩结果页
- `AnalysisView` - 解析查看组件

### 5.2 状态管理
- 当前试卷状态
- 答题进度
- 计时器状态
- 用户答案缓存（防丢失）

### 5.3 本地存储
- 使用 IndexedDB 存储题库数据
- localStorage 缓存答题进度
- 支持离线答题

# SaDuck 公考知识库 - 功能分析

> 网址: https://saduck.top/questionBank/overTheYears.html

## 1. 页面布局和结构

该网站采用 **VitePress** 框架构建，具有典型的文档站点布局：

- **顶部导航栏**：包含站点logo、搜索框和主菜单
- **侧边栏**：分类导航，支持折叠/展开
- **主内容区**：展示具体页面内容
- **页脚**：版权信息和备案号

## 2. 主要功能模块

### 学习内容模块
| 分类 | 内容 |
|------|------|
| 行测 | 资料分析、言语理解、数量关系、判断推理、政治理论、常识 |
| 申论 | 申论与综合应用 |
| 专业 | 公安专业知识 |

### 题库系统
- 📚 历年试卷
- 🎯 专项练习
- 📝 练习记录

### 工具功能

**需登录工具**：
- 生词锦囊
- 计时助手
- 申论解鉴
- 每日打卡

**免登录工具**：
- 词语查询
- 高频词语
- 热榜
- 新闻
- 晨读

## 3. 交互功能

- 🔍 全文搜索（Algolia 驱动）
- 🌓 深浅模式切换
- 📱 响应式菜单
- ⬅️➡️ 页面导航（上一篇/下一篇）
- ⬆️ 返回顶部按钮

## 4. 数据展示方式

采用"考点详细解析，内容通俗易懂"的方式组织知识，支持按主题分类浏览。

## 5. 用户可进行的操作

1. 浏览学习资料
2. 搜索特定内容
3. 进行练习和测试
4. 使用辅助工具
5. 提交反馈（邮箱）
6. 查看更新日志
